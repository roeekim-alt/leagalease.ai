## What it does

Legal.Ease.AI generates seven types of professional commercial agreements in seconds:

| Agreement | Description |
|-----------|-------------|
| **Founders Agreement** | Equity split, vesting (cliff + schedule), BMBY, IP assignment, restrictive covenants |
| **Shareholders Agreement** | Liquidation preference, anti-dilution, ROFR, tag/drag-along, ESOP |
| **Share Purchase Agreement** | M&A deal terms, reps & warranties, escrow, earn-out |
| **Investment Agreement** | SAFE/Series A term sheet, pro-rata, pay-to-play, board rights |
| **Supply Agreement** | Products, pricing, warranties, exclusivity, force majeure |
| **Distribution Agreement** | Territory, commission, marketing obligations, brand guidelines |
| **NDA / Confidentiality** | Mutual or one-way, governing law, duration |

Each agreement generates a **Term Sheet** and a **Full Draft** — both exportable as Word (.docx) or PDF.

## Features

- 🤖 **Frank AI Advisor** — In-app legal AI chat (Claude Sonnet), contextual to the agreement being drafted
- 📄 **Document Intelligence** — Upload existing contracts (PDF/Word/TXT), extract parties, key terms, risks, and get agreement recommendations
- ✏️ **Redline Editor** — Edit drafts with tracked changes, accept/reject redlines, AI-suggested edits
- ⬇️ **Word Export** — Professional .docx with cover page, Arial font, ARTICLE headers, signature blocks
- ⬇️ **PDF Export** — jsPDF-powered, A4, dark ARTICLE headers, clause numbering
- 💾 **Saved Clauses** — Build a library of custom clause templates
- 🔒 **Fully offline** — Single HTML file, no server, no data sent anywhere (except Anthropic API calls you initiate)

## API Key Setup

Legal.Ease.AI uses the **Anthropic Claude API** for:
- Frank AI chat advisor
- Document intelligence analysis
- AI-suggested redlines

Get a free API key at [console.anthropic.com](https://console.anthropic.com).

Enter it in the app via the key icon in the top bar. It's stored only in your browser's localStorage — never sent to any server other than Anthropic's.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| UI | Vanilla HTML/CSS/JS — no framework |
| AI | Anthropic Claude Sonnet (`claude-sonnet-4-20250514`) |
| Word export | Raw OOXML / ZIP (no library) |
| PDF export | jsPDF (CDN, lazy-loaded) |
| Doc parsing | PDF.js + mammoth.js (CDN) |
| Storage | Browser localStorage only |
